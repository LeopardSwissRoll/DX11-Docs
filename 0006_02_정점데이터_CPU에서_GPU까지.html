CPU에서 GPU까지

삼각형 하나를 그리려면 "꼭짓점 3개의 위치와 색"을 GPU에 전달해야 한다. C++ 구조체에서 GPU 메모리까지의 여정.

FVertexColor — 정점 하나의 데이터

C++ 정점 구조체 Common/Info.h
struct FVertexColor
{
    FVertexColor() {}
    FVertexColor(const FVector3D& pos, const FVector4D& color) : _pos(pos), _color(color)
    {
    }
    FVertexColor(float x, float y, float z, float r, float g, float b, float a)
    {
        _pos._x = x; _pos._y = y; _pos._z = z;
        _color._x = r; _color._y = g; _color._z = b; _color._w = a;
    }

    FVector3D _pos;     // 위치 (x, y, z)
    FVector4D _color;   // 색상 (r, g, b, a)
};

정점 하나 = 위치 + 색. FVector3D(12바이트) + FVector4D(16바이트) = 28바이트/정점.

FVertexBuffer / FIndexBuffer

C++ 버퍼 구조체 Common/Info.h
struct FVertexBuffer
{
    ComPtr<ID3D11Buffer> _buffer = nullptr;   // GPU에 있는 버텍스 버퍼
    int32 _size = 0;                           // 정점 한 개당 크기
    int32 _count = 0;                          // 정점의 개수
    std::vector<byte> _data;                   // GPU 전송 전 RAM에 들고 있는 정점 정보
};

struct FIndexBuffer
{
    ComPtr<ID3D11Buffer> _buffer = nullptr;   // GPU에 있는 인덱스 버퍼
    int32 _size = 0;                           // 인덱스 한 개의 크기
    int32 _count = 0;                          // 인덱스의 총 개수
    DXGI_FORMAT _fmt = DXGI_FORMAT_UNKNOWN;    // 데이터 해석 방식
    std::vector<byte> _data;                   // GPU 전송 전 RAM에 들고 있는 인덱스 데이터
};
왜 CPU 사본(_data)을 들고 있나?
GPU에 올린 데이터는 읽어오기 어려워. 나중에 콜리전 체크나 CPU 연산이 필요하면 _data에서 가져다 씀.

NDC (Normalized Device Coordinates)
현재 VS에서 변환 없이 패스스루니까, 좌표는 NDC 기준:
x: -1(왼쪽) ~ +1(오른쪽) / y: -1(아래) ~ +1(위) / z: 0(가까이) ~ 1(멀리)
(0, 0.5, 0) = 화면 중앙에서 위로 절반.

Mesh::CreateMesh() — CPU → GPU

C++ CPU 사본 + GPU 버퍼 생성 Core/Mesh.cpp
bool Mesh::CreateMesh(void* vertexData, int32 size, int32 count, D3D11_USAGE vertexUsage,
    D3D11_PRIMITIVE_TOPOLOGY primitive, void* indexData, int32 indexSize, int32 indexCount,
    DXGI_FORMAT fmt, D3D11_USAGE indexUsage)
{
    _vertexBuffer._size = size;
    _vertexBuffer._count = count;
    _vertexBuffer._data.resize(size * count);
    memcpy(_vertexBuffer._data.data(), vertexData, size * count);

    if (!CreateBuffer(_vertexBuffer._buffer, D3D11_BIND_VERTEX_BUFFER, vertexData, size, count, vertexUsage))
        return false;

    _primitive = primitive;

    if (indexData)
    {
        _indexBuffer._size = indexSize;
        _indexBuffer._count = indexCount;
        _indexBuffer._data.resize(indexSize * indexCount);
        _indexBuffer._fmt = fmt;
        memcpy(_indexBuffer._data.data(), indexData, indexSize * indexCount);

        if (!CreateBuffer(_indexBuffer._buffer, D3D11_BIND_INDEX_BUFFER, indexData, indexSize, indexCount, indexUsage))
            return false;
    }

    return true;
}

CreateBuffer() — D3D11 버퍼 생성

C++ GPU 버퍼 생성 Core/Mesh.cpp
bool Mesh::CreateBuffer(ComPtr<ID3D11Buffer>& buffer, D3D11_BIND_FLAG flag,
    void* data, int32 size, int32 count, D3D11_USAGE usage)
{
    D3D11_BUFFER_DESC bufferDesc = {};
    bufferDesc.ByteWidth = size * count;         // 버퍼 총 크기
    bufferDesc.Usage = usage;                     // DEFAULT, DYNAMIC 등
    bufferDesc.BindFlags = flag;                  // VERTEX_BUFFER or INDEX_BUFFER
    bufferDesc.StructureByteStride = size;

    if (usage == D3D11_USAGE_DYNAMIC)
        bufferDesc.CPUAccessFlags = D3D11_CPU_ACCESS_WRITE;
    else if (usage == D3D11_USAGE_STAGING)
        bufferDesc.CPUAccessFlags = D3D11_CPU_ACCESS_WRITE | D3D11_CPU_ACCESS_READ;

    D3D11_SUBRESOURCE_DATA bufferData = {};
    bufferData.pSysMem = data;

    if(FAILED(Device::Instance().GetDevice()->CreateBuffer(&bufferDesc, &bufferData, buffer.GetAddressOf())))
        return false;

    return true;
}

D3D11_USAGE — CPU↔GPU 접근 방향
DEFAULT
CPU ✕ ← → GPU ✓
GPU 전용. 가장 빠름.
DYNAMIC
CPU → GPU (쓰기만)
자주 업데이트. 파티클 등.
STAGING
CPU ↔ GPU (양방향)
중간 복사 버퍼. 느림.
IMMUTABLE
CPU ✕ → GPU (읽기만)
절대 안 바뀌는 데이터.

Mesh::Render() — 매 프레임

C++ IA 바인딩 + Draw Core/Mesh.cpp
void Mesh::Render()
{
    uint32 stride = _vertexBuffer._size;
    uint32 offset = 0;

    Device::Instance().GetContext()->IASetPrimitiveTopology(_primitive);
    Device::Instance().GetContext()->IASetVertexBuffers(0, 1, _vertexBuffer._buffer.GetAddressOf(), &stride, &offset);

    Device::Instance().GetContext()->IASetIndexBuffer(_indexBuffer._buffer.Get(), _indexBuffer._fmt, 0);

    Device::Instance().GetContext()->DrawIndexed(_indexBuffer._count, 0, 0);
}
